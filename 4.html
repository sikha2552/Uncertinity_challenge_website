Evaluation and Ranking

For evaluating there will be two sections, on one hand, we will evaluate the (contour) dice score for each structure; on the other hand, we will study how the uncertainties are distributed among the different organs.

For the second part, the evaluation of uncertainties will be split in three:

  • Value = 1: The three labelers agree
    • In this case, the uncertainty should be as close as possible to 0 since there is no uncertainty on whether this area corresponds to this structure.
  • Value = 0.67: Two out of the three labelers agree
    • In this case, the uncertainty should be as close as possible to 1 - value (0.33).
  • Value = 0.33: One out of the three labelers agree
    • In this case, the uncertainty should be as close as possible to 1 - value (0.67).
